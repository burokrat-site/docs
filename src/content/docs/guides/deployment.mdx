## Adding new project

<Steps>

1. Click on `Projects` in the left sidebar. Press `+Add` button.

   Name your project e.g. `website`

   You will be redirected to the project's environments page where production environment is already created. You can add development environment later.

2. Click on `production` environment.

   You will be redirected to the resources page. Here you can add new resources such as databases and github repositories.

</Steps>
